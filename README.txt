l'éxecutable ne fonctionne qu'en invite de commande.

Pour le lancer il faut mettre le nom de l'exe suivit du chemin absolu du répertoire qui contient les images à fusionner.

l'image de fin se trouvera dans le répertoire fournit dans la paramètre eu aura pour nom background.jpg.

le programme ne prend que verbose comme deuxième paramètre, celui-ci permet de sauvegarder les fichiers intermédiaires (mask, grayscales,...) dans un répertoire du nom de "files" qui se trouvera dans 
le répertoire fournit dans le première paramètre.
